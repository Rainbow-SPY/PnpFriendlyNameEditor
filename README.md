# PnP FriendlyName Editor

## Description

This project is a simple C# GUI application that allows users to edit the friendly name of a PnP (Patterns and Practices) object. The application prompts the user for the current friendly name and the new friendly name, and then updates the PnP object accordingly.

## How to Use

Just run the application, and it will ask you to enter the current friendly name of the PnP object you want to edit, followed by the new friendly name you want to set. After you provide the necessary information, the application will update the friendly name of the specified PnP object.

Well.. after you run the application, you can view your new device name in devmgmt.msc (Device Manager). Look for the device with the new friendly name you just set.

## Build

you need install .NET 8.0 SDK to build this project. You can download it from the official Microsoft website.

just clone the repository and open the solution file in Visual Studio. Then, build the project to generate the executable.

Have a good time using the application!
